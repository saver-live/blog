为什么在智能标记中使用范围参数

SmartMarkers 中的 range 参数用于在使用 Source（例如 JSON、数据库）的数据填充 Excel 模板时精确控制数据在 Excel 模板中的放置位置和方式。它有助于管理动态数据输出，特别是在处理可变长度数组或复杂分组时。

  1. 控制数据放置并避免重叠：当数据源包含动态数组（例如，每条记录的元素数量不同）时，范围参数可确保数据填充在定义的 Excel 范围内，防止溢出到相邻的单元格或部分。

  2. 处理动态数组公式：对于转置动态数组等作（例如，&=TRANSPOSE（DataArray）），范围参数可确保输出适应实际数据大小，而不会留下先前作的残差值（例如，空字段中的零）。

  3. 支持分组和分层数据：当数据需要分组（例如，嵌套的 JSON 结构）时，range 参数有助于定义分层输出区域。例如，将记录分组到父类别下，而无需手动调整行。如果没有定义的范围，SmartMarker 可能无法准确表示嵌套关系，尤其是在数据源缺乏显式层次结构的情况下。

  4. 改进模板设计和一致性：通过指定范围，用户可以确保格式、公式和边框一致地应用于输出区域。这避免了生成的报告中单元格样式不一致或公式损坏等问题。

  5. 优化性能和数据排序：范围参数允许工具在填充模板之前对数据源进行预排序，确保分组数据以正确的顺序显示。
