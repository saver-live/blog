如何在智能标记中对数据进行分组
2025-09-18 10:32:55 # 报表 #Aspose cell

可能的使用场景

在某些 Excel 报表中，您可能需要将数据分成几组，以便于阅读和分析。将数据分组的主要目的之一是对每组记录运行计算（执行汇总作）。

Aspose.Cells 智能标记允许您按字段对数据进行分组，并在数据集或数据组之间放置汇总行。例如，如果按 Customers.CustomerID 对数据进行分组，则可以在每次组更改时添加摘要记录。

智能标记中的分组数据参数

以下是一些用于对数据进行分组的智能标记参数。

group:normal/merge/repeat

我们支持三种类型的组，您可以选择。

  • normal - 对于列中的相应记录，不会重复分组依据字段值;相反，每个数据组打印一次。
  • merge - 与普通参数的行为相同，不同之处在于它合并每个组集的分组依据字段中的单元格。
  • repeat - 对相应的记录重复分组依据字段值。
    例如：&=Customers.CustomerID（group：merge）

skip

在每组之后跳过指定数量的行。

例如，&=Employees.EmployeeID（group：normal，skip：1）

subtotalN

对与分组依据字段相关的指定字段数据执行汇总作。N 表示介于 1 和 11 之间的数字，这些数字指定计算数据列表中的小计时使用的函数。（1=平均值，2=计数，3=计数，4=最大值，5=最小值,…9=SUM 等）有关更多详细信息，请参阅 Microsoft Excel 帮助中的小计参考。

格式实际上是这样说的： subtotalN：Ref 其中 Ref 引用分组依据列。

例如

  • &=Products.Units（subtotal9：Products.ProductID） 在 Units 字段上指定相对于 Products 表中的 ProductID 字段的汇总函数。
  • &=Tabx.Col3（subtotal9：Tabx.Col1） 在表 Tabx 中按 Col1 对 Col3 字段进行分组指定汇总函数。
  • &=Table1.ColumnD（subtotal9：Table1.ColumnA&Table1.ColumnB） 在表 1 中按列 A 和列 B 对列 D 字段分组指定汇总函数。

如何在智能标记中对数据进行分组

此示例显示了一些正在运行的分组参数。它使用 Northwind.mdb Microsoft Access 数据库并从名为“订单详细信息”的表中提取数据。我们在 Microsoft Excel 中创建一个名为 SmartMarker_Designer.xls 的设计器文件，并将智能标记放入工作表中的各个单元格中。处理标记以填充工作表。数据按组字段放置和组织。

设计器文件有两个工作表。在第一个中，我们放置了带有分组参数的智能标记，如下面的屏幕截图所示。放置三个智能标记（带有分组参数）： &=[订单详情]。OrderID（组：合并，跳过：1）， &=[订单详情]。Quantity（subtotal9：Order Details.OrderID），以及 &=[订单详情]。UnitPrice（subtotal9：Order Details.OrderID） 分别进入 A5、B5 和 C5。

SmartMarker_Designer.xls文件中的第一个工作表，配有智能标记

在设计器文件的第二个工作表中，我们放置了一些更多的智能标记，如下图所示。我们放置以下智能标记：
&=[订单详情]。OrderID（group：normal），
&=[订单详情]。数量
&=[订单详情]。单价，
&=&=B（r）*C（r），以及
&=subtotal9：Order Details.OrderID 分别转换为 A5、B5、C5、D5 和 C6

SmartMarker_Designer.xls文件的第二个工作表，显示混合智能标记。
