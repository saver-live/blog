智能标记参数

Designer 电子表格和智能标记
Designer 电子表格是包含视觉格式、公式和智能标记的标准 Excel 文件。它们可以包含引用一个或多个数据源的智能标记，例如来自项目的信息和相关联系人的信息。智能标记会写入所需信息的单元格中。

所有智能标记都以 &= 开头。数据标记的一个示例是 &=Party.FullName。如果数据标记产生多个项目，例如，一行完整，则后几行会自动向下移动，以便为新信息腾出空间。因此，小计和总计可以放在数据标记后面的行上，以便根据插入的数据进行计算。要对插入的行进行计算，请使用动态公式。

智能标记由大多数信息的数据源和字段名称部分组成。特殊信息也可以通过变量和变量数组传递。变量总是只填充一个单元格，而变量数组可以填充多个单元格。每个单元格仅使用一个数据标记。未使用的智能标记将被删除。

智能标记也可能包含参数。参数允许您修改信息的布局方式。它们作为逗号分隔的列表附加到括号中的智能标记末尾。

智能标记选项

智能标记参数
允许使用以下参数：

  • noadd - 不要添加额外的行来适应数据。
  • skip：n - 每行数据跳过 n 行数。
  • ascending：n 或 descending：n - 对智能标记中的数据进行排序。如果 n 为 1，则该列是排序器的第一个键。处理数据源后对数据进行排序。例如：&=Table1.Field3（ascending：1）。
  • horizontal - 从左到右写入数据，而不是从上到下写入数据。
  • numeric - 如果可能，将文本转换为数字。
  • shift - 向下或向右移动，创建额外的行或列以适合数据。shift 参数的工作方式与 Microsoft Excel 中的工作方式相同。例如，在 Microsoft Excel 中，当您选择单元格范围时，右键单击并选择插入并指定向下移动单元格、向右移动单元格和其他选项。简而言之，shift 参数为垂直/正常（从上到下）或水平（从左到右）智能标记填充相同的函数。
  • copystyle - 将基本单元格的样式复制到该列中的所有单元格。
    参数 noadd 和 skip 可以组合在一起，在交替行上插入数据。由于模板是从下到上处理的，因此应在第一行添加 noadd，以避免在备用行之前插入额外的行。

如果您有多个参数，请用逗号分隔它们，但不要用空格：parameterA，parameterB，parameterC
