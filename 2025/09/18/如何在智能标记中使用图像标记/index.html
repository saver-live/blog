如何在智能标记中使用图像标记
2025-09-18 10:08:14 # 报表 #Aspose cell

可能的使用场景

图像标记是模板系统（如 FoxPro、Handlebars 或现代 UI 框架）中的专用占位符，可将图像或视觉资产动态注入模板。有时，您需要使用智能标记插入图片。Aspose.Cells 可以将图片添加到智能标记中。

智能标记中的图像参数

用于管理图像的智能标记参数。

  • 图片：FitToCell - 自动将图像调整到单元格的行高和列宽。
  • 图片：ScaleN - 将高度和宽度缩放到 N%。
  • Picture：Width：Nin&Height：Nin - 渲染图像高 N 英寸，宽 N 英寸。您还可以指定左侧和顶部位置（以点为单位）。
